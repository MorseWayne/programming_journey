# 入门实战：概览

理论知识是基础，但动手实践才能真正加深理解。本实战系列将为您提供一份详细的分步教程，指导您如何使用 Python 和 `mcp-fastapi-server` 库从零开始构建一个简单的 MCP 服务器。

**我们将构建的服务器具备以下功能：**

1. **一个“工具”（Tool）**：用于执行网络搜索。
2. **一个“资源”（Resource）**：用于读取服务器上本地文件的内容。

最后，我们还将创建一个简单的客户端来与这个服务器交互，以验证其功能是否正常。

2. [Server 开发](./02_server.md)
3. [运行与调试](./03_run.md)
4. [Client 开发与验证](./04_client.md)
